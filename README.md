# WhatsApp Accounting System

A WhatsApp-based accounting system built with Node.js and Baileys.

## Features

- Add transactions
- Record deposits
- View current balance
- Transaction history
- Undo last transaction

## Commands

.done 15x1000
.done 10000
.depo 5000
.total
.history
.undo

## Tech Stack

- Node.js
- Baileys
- JSON Storage

## Installation

npm install

node index.js
